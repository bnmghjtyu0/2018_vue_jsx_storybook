# 2018_vue_jsx_storybook
2018 2月，vue tw 讀書會


## install
```bash
# instal dependncies
npm install
```

## development
```bash
# serve with hot reload at localhost:3000
npm run start

# serve with hot reload at localhost:9001
npm run storybook

# template generator
npm run plop

# jest test
npm run test
```

## production
```bash
# build for production with minification
npm run build

# public storybook
npm run storybook-build
```
